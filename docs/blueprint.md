# **App Name**: CampusLink Library

## Core Features:

- User Authentication: Secure login and registration for mobile app, librarian dashboard, and admin dashboard.
- Book Catalog (Mobile): Browse and search the book catalog on the mobile app.
- Book Details & Reservation (Mobile): View book details and reserve books through the mobile app.
- Inventory Management (Librarian): Manage book inventory, add new books, and track book status.
- Borrow/Return Processing (Librarian): Process book borrowing and returns.
- User Management (Admin): Manage user accounts and roles within the system.
- Reporting (Admin): Generate reports on system usage, book inventory, and other metrics.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a sense of knowledge and trust.
- Background color: Light gray (#F0F2F5) to provide a clean, readable surface.
- Accent color: Muted violet (#7E57C2) to highlight important actions.
- Body font: 'PT Sans' (sans-serif) for readability across all dashboards and mobile app.
- Headline font: 'Space Grotesk' (sans-serif) to create a clear hierarchy on dashboards.
- Use flat, line-style icons for a modern, minimalist aesthetic.
- Maintain a consistent layout across all dashboards and mobile app, prioritizing clear information architecture and user flows.
